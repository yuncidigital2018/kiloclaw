# KiloClaw 文件倉庫

這是 KiloClaw AI 助手的文件倉庫，自動同步所有重要資料。

## 📁 目錄結構

```
.
├── accupass-courses/      # Accupass 課程資料
├── daily-monitor/         # 每日監控腳本與記錄
├── guoben-academy/        # 國本學堂課程資料
├── scripts/               # 公用腳本
├── system-configs/        # 系統設定檔案
└── README.md              # 本檔案
```

## 🔄 自動同步

所有 `.md` 和 `.json` 檔案會自動從工作區同步到這個倉庫。

## 📝 文件類型

- **Markdown (.md)**: 報告、課程清單、說明文件
- **JSON (.json)**: 結構化資料、設定檔案
- **Shell (.sh)**: 自動化腳本
