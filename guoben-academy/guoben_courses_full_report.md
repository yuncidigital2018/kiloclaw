### 課程 1：跟植物診療師學診斷-小番茄/小黃瓜健康管理實戰

| 項目 | 內容 |
|------|------|
| **日期** | 2026/02/06 (五) 13:00-16:30 及 02/11 (二) 13:00-16:30 |
| **地點** | 創新學院203教室（嘉義縣太保市祥和一路東段1號） |
| **講師** | 郭章信 教授（中興大學植醫學程） |
| **招生人數** | 實體60人，線上100人 |
| **報名截止** | 2026/02/03 (二) 23:59 |

**課程重點：**
- 小番茄、小黃瓜常見病徵案例實戰解析
- 真菌、細菌、生理障礙如何區分
- 全台實戰案例：從誤判到正確診斷
- 學會與植物診療師有效溝通

**課程說明：**
明明噴藥了，為什麼還是沒效？看起來像疫病，檢驗卻說不是？到底是缺肥、生病還是環境出問題？郭章信教授走遍各地溫室，帶你學會從病徵觀察→建立診斷邏輯→精準判斷病因。不再誤判、不再白噴藥，用植物診療師的眼光管理你的溫室！
